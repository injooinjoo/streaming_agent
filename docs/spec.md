# Weflab Clone 상세 기능 명세서 (Specification)

이 문서는 스트리밍 에이전트 개발을 위한 핵심 기능 및 UI/UX 요구사항을 정리한 가이드라인입니다.

## 1. 공통 사항 (Common)

- **디자인 컨셉**: ChatGPT 스타일 (다크 모드, 프리미엄 그린 포인트, 깔끔한 사이드바, Inter 폰트).
- **기술 스택**: React (Vite), Express, Socket.io, SQLite3, Lucide-React.
- **오버레이 관리**: 각 기능별 전용 URL 제공 (복사/새로고침/초기화).

---

## 2. 채팅창 (Chat Overlay)

### 기본 설정

- **테마**: 기본, 탄막, 고양이, 설날, 롤, 스타, 배그, 하트, 겨울, 레트로, 무지개, 크레용, 골드, 점선, 윈도우, 카카오톡, 라운드, 풍선, 칠판, 네온, 박스, 가죽, 포스트잇, 음식, 오버워치 등.

### 표시 및 효과

- **정렬**: 한줄/줄바꿈, 닉네임 구분선, 시작라인/개별라인, 좌/우/가운데 정렬.
- **애니메이션**: Slide, Fade, Bounce, Flip, Rotate, Zoom, Tada, Wobble 등 다양한 효과 지원.
- **상세**: 투명도 (0-100%), 툴팁/미리보기 사용 여부, 커스텀 CSS.

### 상세 기능

- **표시 항목**: 플랫폼 아이콘, 스크롤 사용, 닉네임/ID 표시, 랜덤 색상.
- **필터링**: 사용자, 후원 알림, 봇, 단어 필터링 (♡ 처리).
- **알림**: 입장/채팅 알림 (텍스트/알림음/음성).

### 내장 위젯 (명령어 지원)

- **총 시청자수**: 상/하 위치 설정.
- **공지 (Notice)**: 테마 지원, 명령어(`!공지`, `!공지삭제`, `!공지 시간` 등).
- **타이머 (Timer)**: 현재시간/업타임, 명령어(`!시간`, `!시간 10분`, `!시간/추가` 등).
- **채팅 고정 (Pin)**: 특정 채팅 상단 고정 및 음성(TTS) 읽기.

---

## 3. 후원 알림 (Alert Overlay)

### 기본 설정

- **테마**: 기본, 하트모션, 스타모션, 고양이, 설날, 롤/스타/배그 등 채팅창과 유사한 테마군 지원.
- **창 설정**: 투명도, 툴팁, 커스텀 CSS, 일시중지 표시.

### 작동 규칙

- **플랫폼 (SOOP)**: 별풍선, 애드벌룬, 영상풍선, 미션, 스티커 선택적 사용.
- **우선순위**: 개수 중첩 시 마지막 설정 우선.
- **범위**: 전체 범위 설정은 리스트 최상단 권장.

### 효과 및 사운드

- **텍스트 효과**: Flash, Bounce, Tada, Flip, Fade, Rotate, RollIn 등.
- **알림 효과**: 등장/퇴장 개별 애니메이션 설정.
- **TTS**: 볼륨, 속도, 글자수 제한. GIF 이미지 초기화 지원.

---

## 4. 후원 자막 (Donation Subtitles)

- **로직**: 최대 1주일(시작일 설정 시 1달) 데이터 로드 및 복구 지원.
- **모드**: 개수, MVP, 랭킹(메달 표시), 최근, 후원 이미지.
- **디자인**: 투명도, 커스텀 CSS, 플랫폼 구분 아이콘, 익명 표시 필터링.

---

## 5. 목표치 그래프 (Goal Graph)

- **대상**: 별풍선, 두산, 시청자 수.
- **모양**: 바(Bar), 원, 하트, 별, 반원.
- **기능**: 시작/최대값, 자동 증가, 반복 효과, 완료 이펙트, 시작일 집계.

---

## 6. 전광판 (Ticker)

- **기능**: 메시지 흐름 표시, 남은 시간 표시, 숨김 모드.
- **권한**: 매니저/열혈 등 등급별 사용 제한.
- **명령어**: `!전광판`, `!전광판삭제`, `!전광판숨김` 등.

---

## 7. 기타 위젯 (자막 & 배너)

- **자막**: 일반 텍스트, 점수판(스타/박스/멀티). 키워드 지원(`{시간}`, `{날짜}`, `{요일}`, `{초}`).
- **배너**: 이미지 슬라이드/페이드 전환. 배너 표시/전환 시간 설정.

---

## 8. 디자인 커스텀 (공통)

- **아이콘**: SOOP 등급별 아이콘 설정.
- **폰트**: G마켓 산스, 나눔고딕, 쿠키런, 배달의민족 등 50종 이상 지원 + 웹폰트.
- **배경**: 배경색, 이미지 업로드(드래그 앤 드롭), 외부 링크, 채우기 옵션.
